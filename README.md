# Discord-bot

# 🎵 Discord Music Bot

Um bot de música simples e divertido para Discord, feito em Python com `discord.py` e `yt-dlp`.

---

## ✨ Funcionalidades

- **Tocar músicas e playlists** do YouTube
- Suporte a filas (queue)
- Comandos de controle: `skip`, `stop`, `mostrar`
- Verificação de canal específico
- Interface limpa com embeds e emojis
- Respostas em português (bem humoradas)

---

## 📋 Comandos

| Comando       | Descrição                              |
|---------------|----------------------------------------|
| `/play [url]` | Toca uma música ou playlist do YouTube |
| `/skip`       | Pula para a próxima música             |
| `/mostrar`    | Mostra a fila de músicas atual         |
| `/stop`       | Para a música e desconecta o bot       |
| `/ping`       | Mostra a latência do bot               |
| `/help`       | Mostra esta lista de comandos          |
